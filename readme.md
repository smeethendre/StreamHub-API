# Video Platform Backend API

A backend-only, production-style video platform built with **Node.js + Express + MongoDB + Multer + Cloudinary**.  
Designed to showcase advanced backend engineering: secure auth, uploads, scalability patterns, queues/workers, observability, and deployment.


## Tech Stack

**Current (required):**
- Node.js, Express.js
- MongoDB + Mongoose
- Multer
- Cloudinary
- JWT
